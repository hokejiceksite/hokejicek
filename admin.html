Nejčastější problémy a řešení
  • Zkontrolujte, že jste v souboru ponechal(a) přesný firebaseConfig (nejlépe copy/paste z konzole).
  • V Firebase Console → Project settings → Authorized domains přidejte: localhost, hokejicek.site a VAŠE_UZIV_JMENO.github.io (pokud používáte Pages).
  • V Realtime Database → Rules nastavit například: 
    { "rules": { ".read": true, ".write": "auth != null" } }
  • Pokud dostanete chybu typu api-key-not-valid, zkontrolujte, že v configu máte i databaseURL a že config je přesně z konzole.
